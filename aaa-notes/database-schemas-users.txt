2.4. Create User Accounts in PostgreSQL
=======================================

The following outlines the steps to take when setting up the database in PostgreSQL.

1. Log in to PostgreSQL using psql.
2. Create the schema using a create schema command like:
   create schema i2b2demodata
3. The following table lists the schemas to be created:

Database
i2b2demodata
i2b2hive
i2b2metadata
i2b2pm
i2b2workdata

4. Create the users.
a. create user i2b2demodata password 'demouser';
b. grant all on all tables in schema i2b2demodata to i2b2demodata;

User Name		Password 	Database
i2b2demodata 	demouser 	i2b2demodata
i2b2hive 		demouser 	i2b2hive
i2b2metadata 	demouser 	i2b2metadata
i2b2pm 			demouser 	i2b2pm
i2b2workdata 	demouser 	i2b2workdata
